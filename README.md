iemop
=====

基于冒泡开放API，实现的个性化定制版冒泡。

如何使用
=======

i冒泡是一个java开放的web客户端。运行是需要本地安装jre1.6以上。


配置微博发送的几个核心概念。

1. 微博账号
2. 内容库 -- i冒泡会自动同步冒泡的内容库，到你的个人内容库。也可以手动导入自己的内容库。
3. 定时计划 -- 定义微博发送的时间点，和用户组，每个时间点都需要指定微博模版。
4. 微博模版 -- 定义微博的发送方式，内容分组，自动文案放的位置等。
5. 模版短语 -- 在发送微博时，自动加入到文案前后的内容。
