数据导入功能

为了方便批量数据管理，目前所有数据操作功能，都是通过上传文本文件，导入数据的方式完成。

导入数据格式

导入数据的字段分隔符统一都是英文的分号';'。如果字段没有值，可以用空格代替。文件里面每行就是一条记录。不需要标题行

微博账号格式

微博应用;登陆名;登陆密码;淘宝PID;用户标签1,用户标签2

内容库格式

内容来源;类容标签;文本内容;图片地址;商品ID;冒泡库ID;

定时计划格式

计划名;用户组;任务状态(0|1)
时间点;内容分组;内容标签;发布模版ID

发布模版格式

模版名;发送应用;短网址域名;是否放到评论;内容模版;评论模版

模版短语格式

分组名;短语
